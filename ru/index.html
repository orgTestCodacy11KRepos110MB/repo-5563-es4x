ES for Eclipse Vert.x

A Modern JavaScript runtime for Eclipse Vert.x

Как начать →

Главное - простота

Сфокусируйтесь на вашем коде, а не на настройках проекта npm.

Сила Vert.x

Наслаждайтесь масштабируемостью и производительностью Vert.x, используйте компоненты Reactive Vert.x для JavaScript, и пишите на JavaScript или TypeScript.

Производительность

ES4X работает поверх GraalVM, предлагая высокую производительность для JavaScript приложений на уровне Java или даже быстрее.

# Просто, как 1, 2, 3

# (1) создаем
npm init @es4x project

# (2) устанавливаем зависимости
npm install # OR yarn

# (3) запускаем
npm start # OR yarn start

Примечание о совместимости!

Для ES4X требуется GraalVM (opens new window) или Java >= 8. Если вы не знаете, какая версия установлена на вашей системе, попробуйте воспользоваться jabba (opens new window).
