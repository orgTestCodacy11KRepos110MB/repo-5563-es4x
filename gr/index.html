ES για Eclipse Vert.x

Ένα σύγχρονο JavaScript runtime για το Eclipse Vert.x

Get Started →

Απλότητα πρωτα απο ολα

Η ελάχιστη ρύθμιση με δομή έργου με επίκεντρο το npm σάς βοηθά να εστιάσετε στον κώδικά σας.

Vert.x-Powered

Απολαύστε επεκτασιμότητα και εμπειρία απόδοσης Vert.x, χρησιμοποιήστε στοιχεία Reactive Vert.x σε JavaScript και αναπτύξτε με JavaScript ή TypeScript.

Αποτελεσματικός

Το ES4X τρέχει πάνω από το GraalVM προσφέροντας εξαιρετική απόδοση για JavaScript εφαρμογές ισοδύναμες ή καλύτερες από την Java.

# Τόσο εύκολο όσο 1, 2, 3

# (1) δημιουργία
npm init @es4x project

# (2) εγκατάσταση εξαρτήσεων
npm install # OR yarn

# (3) εκτέλεση
npm start # OR yarn start

ΣΗΜΕΙΩΣΗ ΣΥΜΒΑΤΟΤΗΤΑΣ

Tο ES4X απαιτεί GraalVM (opens new window) ή Java >= 8. Εάν δεν είστε βέβαιοι ποια έκδοση είναι εγκατεστημένη στο σύστημά σας, μπορείτε να χρησιμοποιήσετε jabba (opens new window).
