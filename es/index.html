ES para Eclipse Vert.x

Un JavaScript runtime moderno para Eclipse Vert.x

Get Started →

Simplicidad primero

Configuracion minima con una estructura de proyecto centrada en npm te ayuda a concentrarte en tu codigo.

Desarrollado con Vert.x

Disfruta de la escalabilidad y rendimiento de Vert.x, usa componentes Reactive Vert.x en JavaScript, y desarolla con JavaScript o TypeScript.

Rendimiento

ES4X funciona encima de GraalVM ofreciendo un gran rendimiento en aplicaciones JavaScript a la par o mejor que en Java.

# Tan facil como 1, 2, 3

# (1) Crea
npm init @es4x project

# (2) instala las dependencias
npm install # OR yarn

# (3) Ejecuta
npm start # OR yarn start

NOTA DE COMPATIBILIDAD

ES4X necesita GraalVM (opens new window) o Java >= 8. No estas seguro de que version tienes instalada en tu sistema, considera usar jabba (opens new window).
